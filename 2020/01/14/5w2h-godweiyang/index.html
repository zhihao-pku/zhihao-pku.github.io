案例1：一个人的受教育程度（E）是否会影响工资（D）

案例2：基因型1或2（E）是否会影响心肌梗死发生（D）

第一种情况（案例1）：

  • A1为每日体育锻炼，A1与受教育程度和工资均没有关联
  • 无需调整A1；若调整A1，调整后的E-D关联仍等于边际关联，但调整后的E-D关联比边际关联的方差大
# 比较调整无关变量x2后x1估计方差的变化：变大
library(tidyverse)

set.seed(123)
x1 <- rnorm(100, 0, 1)
x2 <- rnorm(100, 0, 5)
y <- 2 + 2 * x1 + rnorm(100, 0, 2)

df <- data.frame(x1 = x1, x2 = x2, y = y)

ggplot(data = df, aes(x = x1, y = y)) +
  geom_point()

fit <- lm(y ~ x1, data = df)
sigma(fit)^2

fit <- lm(y ~ x1 + x2, data = df)
sigma(fit)^2

第二种情况（案例1）：

  • A2为其父母受教育程度，A2会影响其受教育程度，但A2不会影响其工资
  • 如果存在一混杂变量如个人能力，个人能力同时会导致受教育程度和工资增加，则可使用A2作为工具变量，进行工具变量回归
  • 无需调整A2；若调整A2，调整后的E-D关联仍等于边际关联，但调整后的E-D关联比边际关联的方差大

第三种情况（案例2）：

  • A3为眼皮单双，基因型会影响眼皮单双，但眼皮单双与心肌梗死发生无关
  • 无需调整A3；若调整A3，调整后的E-D关联仍等于边际关联，但调整后的E-D关联比边际关联的方差大

第四种情况（案例2）：

  • A4为性别，不同性别中基因型与心肌梗死的关联不一致

第五种情况（案例2）：

  • A5为单路径碰撞变量（$D \rightarrow A$、$E \rightarrow D \rightarrow A$）；调整A5，如固定A5于某一水平
  • A5为死亡。在一项横截面研究中，调查基因型1和基因型2心肌梗死发生率。因发生重度心肌梗死会死亡，调查到的人为未发生心肌梗死和轻度心肌梗死者（固定A5于幸存者）。观察到的$RR_{obs} = \frac{\frac{10}{60}}{\frac{10}{60}} = 1$，真实的$RR_{act}=\frac{\frac{12}{62}}{\frac{18}{68}}=0.73$
  • 无需调整A5；若调整A5，产生单路径碰撞偏移（uni-path collider bias）
轻度心肌梗死 重度心肌梗死 未发生心肌梗死
基因型1 10 2 50
基因型2 10 8 50

第六种情况（案例1）：

  • A6为个人能力
  • 需调整A6

第七种情况（案例1）：

  • A7为求婚是否成功。受教育程度和工资都会导致求婚成功；若固定A7，比如在求婚未成功的人群中，受教育程度和工资会出现额外的负向关联
  • 无需调整A7；若调整A7，产生双路径碰撞偏移（bi-path collider bias）

第八种情况（案例1）：

  • A8为找工作的技能。受教育程度高使得找找工作的技能提升，进而导致工资高
  • 无需调整A8；A8为E-D效应的中介，若调整A8，则得到E-D间的直接效应

第九种情况（案例1）：

  • A9为成人教育。循环因果问题：受教育程度高导致工资高，工资高后有额外的收入参加成人教育，使得受教育程度进一步提高
  • 如何得到E-D的效应那？工具变量回归：选择其父母受教育程度作为工具变量（假设：父母受教育程度会影响其受教育程度，但父母受教育程度不会影响其工资）

$\hat \beta_j$的方差取决于三部分：

  • 误差方差$\sigma^2$

  • $x_j$的样本变异$SST_j$：增加样本容量有助于增加$x_j$的方差

  • $x_j$与其他变量间的线性关系$R_j^2$：减小变量$x_j$与纳入变量的多重共线性

概念

碰撞变量：

  • 路径上的一个变量存在两个及以上的母变量或祖先变量

开放or关闭路径：

  • 开放路径：路径上不存在碰撞变量

  • 关闭路径：路径上存在碰撞变量

有向or无向路径

所有的有向路径均为开放路径

$A \rightarrow B \rightarrow C$

开放路径包括有向路径和无向路径

$A \rightarrow B \rightarrow C$、$A \leftarrow B \rightarrow C$

关闭路径均为无向路径

$A \rightarrow B \leftarrow C$

collider bias

That conditioning on a common effect of exposure and outcome may cause selection, or collider-stratification, bias is not intuitive.

d-separation（direct-separation）：有向分离

$E \rightarrow A7 \leftarrow D$（head to head）

  • 为关闭路径，A7不影响E-D关联，不应调整；但在病例对照研究中，根据暴露和结局选择研究人群，相当于对A7进行调整，从而导致collider偏倚
  • E和D边际独立；给定A7后，E和D条件相关（conditional dependent）；如下图，调整Z后，X和Y出现关联（collider bias）；可能的DAG有1种：

$E \leftarrow A6 \rightarrow D$（tail to tail）

  • 为开放路径，A6会影响E-D关联

  • E和D边际相关（marginal association）；给定A6后，E和D条件独立；如下图，$X \perp Y|Z$；可能的DAG有3种：

library(dagitty)
library(ggdag)

theme_set(theme_dag_gray_grid())

dag <- dagitty("dag {
X -> Z -> Y -> V
X -> W -> U
Y -> W
               }")

coordinates(dag) <- list(
  x = c(X = 1, Z = 2, Y = 3, V = 4, W = 2, U = 4),
  y = c(X = -1, Z = -2, Y = -1, V = -2, W = 0, U = 0)
)
plot(dag)

Causal的三个层次

from association (seeing) to intervention (doing) to counterfactuals (imagining).

研究X与Y关系时，若没有关于Z与X和Y关系的先验，根据数据发现：

  • X和Y边际相关，但condition on Z后，X和Y独立，则可出现左上3幅图。我们没法判断是3种关系的哪一种（中介or混杂引起的边际相关）
  • 在RCT中，X是人为指定的：
    • 左下1中：改变X会影响Y，X是Y的因
    • 左下2-3以及右下图，改变X均不会影响Y，X不是Y的因

DAG中哪些变量需要调整

E是暴露，D是结局，X1、X2、X3是其他变量

最后一幅DAG，E-D可能的路径有：

$E \leftarrow X3 \leftarrow X2 \rightarrow D$：调整X2或X3均可

$E \leftarrow X2 \rightarrow D$：调整X2

$E \leftarrow X2 \leftarrow X1 \rightarrow D$：调整X1或X2

$E \rightarrow D$
