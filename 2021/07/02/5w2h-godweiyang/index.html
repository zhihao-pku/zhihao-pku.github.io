多变量线性回归和Logistic回归中绘制一个自变量与Y或预测概率的回归线

回归
回归
发布日期:   2021-07-02

Welcome !!

  • 多变量回归中有多个自变量，要绘制其中一个自变量与结局的关系时，要把其他自变量固定在某一个水平
  • 一般而言，连续变量固定在均值水平，分类变量固定在对照水平，也可以自行指定对照水平
  • 以上三种情况，结果中都应该注明其他变量固定的水平

绘制方法：

  1. 简单：使用ggeffects包
  2. 灵活：手工计算绘图数据并绘图
library(ggplot2)
library(dplyr)
library(ggeffects)
library(splines)
library(titanic)
library(lmerTest)
rm(list = ls()) # 清空工作空间数据！！！

线性回归：预测均值

方法一：使用ggeffects包

稍微复杂点的例子

mpg以三次多项式形式进入模型：
$$\hat wt = \beta_0 + \beta1 \times mpg + \beta_2 \times mpg^2 + \beta_3 \times mpg^3 + \beta_4 \times disp + \beta5 \times vs \
=\beta_0 + \beta1 \times (mpg + \frac{\beta_2}{\beta_1} \times mpg^2 + \frac{\beta_3}{\beta_1} \times mpg^3) + \beta_4 \times disp + \beta5 \times vs$$
此时mpg增加n个单位（相比于0），wt增加$ n+ \frac{\beta_2}{\beta_1} \times n^2 + \frac{\beta_3}{\beta_1} \times n^3$个$\beta_1$，因此wt随mpg呈非线性变化关系

使用mpg [all]同时指定mpg的所有项:mpg + I(mpg^2) + I(mpg^3)

fit <- lm(wt ~ mpg + I(mpg^2) + I(mpg^3) + disp + vs, data = data)
ggpredict(fit, terms = "mpg [all]") %>% 
  plot()

mpg以立方样条形式进入模型；bs()函数默认不含拐点，此时相当于以三次多项式形式进入模型

fit <- lm(wt ~ bs(mpg) + disp + vs, data = data)
ggpredict(fit, terms = "mpg") %>% 
  plot()

mpg以自然立方样条形式（即限制立方样条）进入模型；ns()函数默认不含拐点，同时由于自然样条被限制为前后两端为线性形式，所以这里相当于线性形式进入模型

fit <- lm(wt ~ ns(mpg) + disp + vs, data = data)
ggpredict(fit, terms = "mpg") %>% 
  plot()

mpg以分段线性形式进入模型

fit <- lm(wt ~ mpg + I((mpg > 20) * (mpg - 20)) + disp + vs, data = data)
ggpredict(fit, terms = "mpg") %>% 
  plot() + 
  geom_vline(xintercept = 20, linetype = 2, color = "red")

方法二：手工绘制

mpg以三次多项式形式进入模型为例

fit <- lm(wt ~ mpg + I(mpg^2) + I(mpg^3) + disp + vs, data = data)

绘图时实际绘出的为折线图，即找到回归线上的各个点，然后连接起来便得到了回归线

# X轴mpg数据，每间隔0.1取一个点
mpg <- seq(min(data$mpg), max(data$mpg), 0.1)

# disp固定在均值水平，vs固定在对照水平
dt <- data.frame(mpg = mpg, 
                 disp = mean(data$disp), 
                 vs = factor(0, levels = c(0, 1)))

# 固定disp和vs后，wt预测均值随mpg的变化
dt$pred <- predict(fit, newdata = dt)

预测均值标准误计算

$X \times \beta$标准误计算公式为：
$$\sqrt{diag(X\times vcov(fit) \times t(X))} \tag{1}$$
$X$为自变量设计矩阵，$vcov(fit)$为回归系数协方差矩阵，$t(X)$代表$X$转置
根据公式1计算给定mpg、disp和vs后，wt预测均值标准误及其95% CI

mm <- model.matrix(~ mpg + I(mpg^2) + I(mpg^3) + disp + vs, data = dt)
dt$se <- sqrt(diag(mm %*% vcov(fit) %*% t(mm))) # 标准误

dt$lwr <- dt$pred - qnorm(0.975) * dt$se # 95%CI 下限
dt$upr <- dt$pred + qnorm(0.975) * dt$se # 95%CI 上限

绘制出disp固定在均值水平，vs固定在对照水平，wt随mpg变化的回归线及其95% CI

ggplot(data = dt, aes(x = mpg)) + 
  geom_line(aes(y = pred), linetype = 2, color = "red") + 
  geom_line(aes(y = lwr), linetype = 2, color = "red") + 
  geom_line(aes(y = upr), linetype = 2, color = "red")

对比可知，手工绘制与ggeffects包绘制出的图形完全一致

ggpredict(fit, terms = "mpg [all]") %>% 
  plot() + 
  geom_line(data = dt, aes(x = mpg, y = pred), linetype = 2, color = "red") + 
  geom_line(data = dt, aes(x = mpg, y = lwr), linetype = 2, color = "red") + 
  geom_line(data = dt, aes(x = mpg, y = upr), linetype = 2, color = "red")

预测均值标准误的简单计算

可以不使用公式1计算标准误，直接通过predict()获取

fit <- lm(wt ~ mpg + I(mpg^2) + I(mpg^3) + disp + vs, data = data)

mpg <- seq(min(data$mpg), max(data$mpg), 0.1)
dt <- data.frame(mpg = mpg, 
                 disp = mean(data$disp), 
                 vs = factor(0, levels = c(0, 1)))
# 获取预测均值
dt$pred <- predict(fit, newdata = dt, se.fit = T)$fit
# 获取预测均值标准误
dt$se <- predict(fit, newdata = dt, se.fit = T)$se.fit
dt$lwr <- dt$pred - qnorm(0.975) * dt$se
dt$upr <- dt$pred + qnorm(0.975) * dt$se

ggpredict(fit, terms = "mpg [all]") %>% 
  plot() + 
  geom_line(data = dt, aes(x = mpg, y = pred), linetype = 2, color = "red") + 
  geom_line(data = dt, aes(x = mpg, y = lwr), linetype = 2, color = "red") + 
  geom_line(data = dt, aes(x = mpg, y = upr), linetype = 2, color = "red")

Logistic回归：预测概率

以titanic包中的泰坦尼克号生存数据为例，Survived为0、1，1代表存活

data <- titanic_train
data <- na.omit(data[, c("Survived", "Age", "Sex", "Pclass")])
str(data)
## 'data.frame':    714 obs. of  4 variables:
##  $ Survived: int  0 1 1 1 0 0 0 1 1 1 ...
##  $ Age     : num  22 38 26 35 35 54 2 27 14 4 ...
##  $ Sex     : chr  "male" "female" "female" "female" ...
##  $ Pclass  : int  3 1 3 1 3 1 3 3 2 3 ...
##  - attr(*, "na.action")= 'omit' Named int [1:177] 6 18 20 27 29 30 32 33 37 43 ...
##   ..- attr(*, "names")= chr [1:177] "6" "18" "20" "27" ...
data$Sex <- factor(data$Sex, levels = c("male", "female"))
data$Pclass <- factor(data$Pclass, levels = c(1, 2, 3))

方法一：使用ggeffects包

Age以线性形式进入模型，绘制生存概率随Age变化的回归线；Sex固定为male，Pclass固定为1

fit <- glm(Survived ~ Age + Sex + Pclass, 
           data = data, family = binomial())

ggpredict(fit, terms = "Age")
## Data were 'prettified'. Consider using `terms="Age [all]"` to get smooth plots.
## # Predicted probabilities of Survived
## # x = Age
## 
##  x | Predicted |       95% CI
## -----------------------------
##  0 |      0.78 | [0.63, 0.88]
## 10 |      0.71 | [0.57, 0.81]
## 20 |      0.63 | [0.51, 0.73]
## 30 |      0.54 | [0.44, 0.63]
## 40 |      0.44 | [0.36, 0.54]
## 50 |      0.36 | [0.27, 0.45]
## 60 |      0.28 | [0.19, 0.38]
## 80 |      0.15 | [0.08, 0.27]
## 
## Adjusted for:
## *    Sex = male
## * Pclass =    1
ggpredict(fit, terms = "Age") %>% plot()
## Data were 'prettified'. Consider using `terms="Age [all]"` to get smooth plots.

Age以二次多项式形式进入模型绘制回归线；Age的其他形式以此类推

fit <- glm(Survived ~ Age + I(Age^2) + Sex + Pclass, 
           data = data, family = binomial())
ggpredict(fit, terms = "Age [all]") %>% plot()

方法二：手工绘制

Logistic回归表达式

$$
logit(P)=log(\frac{P}{1-P})=X \times \beta \tag{2}
$$

预测概率
$$
P=\frac{e^{X \times \beta}}{1 - e^{X \times \beta}} \tag{3}
$$

由于预测概率并不服从正态分布，因此求预测概率及其95% CI时，要先求出线性预测值$X \times \beta$（Xbeta，服从正态分布）及其95% CI，而后使用公式3转换成预测概率及其95% CI

Age以二次多项式形式进入模型为例

fit <- glm(Survived ~ Age + I(Age^2) + Sex + Pclass, 
           data = data, family = binomial())
# X轴Age数据，每间隔0.1取一个点
Age <- seq(min(data$Age), max(data$Age), 0.1)

# Sex和Pclass分别固定在male和1
dt <- data.frame(Age = Age, 
                 Sex = factor("male", levels = c("male", "female")), 
                 Pclass = factor(1, levels = c(1, 2, 3)))

# 固定Sex和Pclass后，线性预测值Xbeta随Age的变化
dt$pred <- predict(fit, newdata = dt, type = "link")

按照前述线性回归公式1计算出线性预测值Xbeta的标准误及其95% CI

mm <- model.matrix(~ Age + I(Age^2) + Sex + Pclass, data = dt)
dt$se <- sqrt(diag(mm %*% vcov(fit) %*% t(mm)))

dt$pred_lwr <- dt$pred - qnorm(0.975) * dt$se
dt$pred_upr <- dt$pred + qnorm(0.975) * dt$se

Xbeta转换成预测概率P

dt$prob <- exp(dt$pred) / (1 + exp(dt$pred))
dt$prob_lwr <- exp(dt$pred_lwr) / (1 + exp(dt$pred_lwr))
dt$prob_upr <- exp(dt$pred_upr) / (1 + exp(dt$pred_upr))

绘制出预测概率P及其95% CI随Age变化的回归线

ggplot(data = dt, aes(x = Age)) + 
  geom_line(aes(y = prob), linetype = 2, color = "red") + 
  geom_line(aes(y = prob_lwr), linetype = 2, color = "red") + 
  geom_line(aes(y = prob_upr), linetype = 2, color = "red")

对比可知，手工绘制出的图形跟ggeffects绘制出的图形完全一致

ggpredict(fit, terms = "Age [all]") %>% 
  plot() + 
  geom_line(data = dt, aes(x = Age, y = prob), linetype = 2, color = "red") + 
  geom_line(data = dt, aes(x = Age, y = prob_lwr), linetype = 2, color = "red") + 
  geom_line(data = dt, aes(x = Age, y = prob_upr), linetype = 2, color = "red")

明白了绘制原理，其他模型绘制的方法也类似；比如绘制线性混合模型回归线

以lmerTest包中的sleepstudy数据集为例；新创建一个Sex变量，拟合多变量回归

方法二：手工绘制

Days <- seq(min(data$Days), max(data$Days), 0.1)
dt <- data.frame(Days = Days, 
                 Sex = factor(0, levels = c(0, 1)))
# re.form = NA指定获取固定效应均值
dt$pred <- predict(fit, newdata = dt, re.form = NA)

mm <- model.matrix(~ Days + Sex, data = dt)
dt$se <- sqrt(diag(mm %*% vcov(fit) %*% t(mm)))

dt$lwr <- dt$pred - qnorm(0.975) * dt$se
dt$upr <- dt$pred + qnorm(0.975) * dt$se

ggplot(data = dt, aes(x = Days)) + 
  geom_line(aes(y = pred), linetype = 2, color = "red") + 
  geom_line(aes(y = lwr), linetype = 2, color = "red") + 
  geom_line(aes(y = upr), linetype = 2, color = "red")

对比可知，手工绘制出的图形跟ggeffects绘制出的图形一致

ggpredict(fit, terms = "Days") %>%
